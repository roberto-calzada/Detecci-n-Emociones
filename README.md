Reconocimiento de Emociones con IA

Modelo de deep learning para detectar emociones faciales a partir de imágenes.

Tecnologías
Python
TensorFlow / Keras
OpenCV
Google Colab

 Descripción

Este proyecto implementa una red neuronal capaz de clasificar emociones humanas (feliz, triste, enojo, etc.) usando imágenes faciales.

Dataset

Dataset utilizado: CREMA-D Video
Link: https://www.kaggle.com/datasets/stefanogiannini/crema-d-video?select=1001_IEO_ANG_LO.flv

Proceso
Preprocesamiento de imágenes
Entrenamiento del modelo
Evaluación de precisión
Predicción de emociones

Resultados

Acurracy: ~90%

Cómo usarlo

Abrir el notebook
Ejecutar celdas en orden
Cargar imagen para predicción

Notas

Proyecto desarrollado como práctica de visión por computadora e inteligencia artificial.
